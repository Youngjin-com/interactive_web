Noto Sans는 Google이 전 세계의 언어를 지원하기 위해 만든 폰트 패밀리 중 하나입니다. "Noto"라는 이름은 "No more Tofu"의 약자로, 'tofu'는 알 수 없는 문자를 나타내는 네모 상자를 의미합니다. 따라서 Noto 프로젝트의 목표는 모든 언어의 모든 문자를 표현하는 것입니다.
